XICRA.config.set_config.access_check(fn, mode=1)[source]¶

Check executable permission

This function checks whether a given path is a folder or file and if it is executable and accessible. It also works if a java jar file provided.

Parameters

  • fn (string) – Absolute path file

  • mode (string) – Value to pass as the mode parameter of access()

mode defaults to:

  • os.F_OK: Value to pass as the mode parameter of access() to test the existence of path.

  • os.X_OK: Value to include in the mode parameter of access() to determine if path can be executed.

We modified the code to work if java jar files provided.

XICRA.config.set_config.check_dependencies(Debug)[source]¶

Check if available the different software required for XICRA execution.

Using the function XICRA.config.extern_progs.read_dependencies() the information for all the dependencies is retrieved from file XICRA/config/software/dependencies.csv.

For each software, the function XICRA.config.set_config.get_exe() retrieves whether it is installed in the system or not and its version and it is check using XICRA.config.set_config.check_install().

Debug (boolean) – True/False for debugging messages

Returns

Print messages and information

XICRA.config.set_config.check_python_packages(Debug)[source]¶

This functions checks whether the packages installed in the system fulfilled the minimum version specified in the configuration folder.

It uses function XICRA.config.set_config.get_python packages() to retrieve the version of the python packages installed in the system. Then it uses XICRA.config.extern_progs.min_python_module_version() to retrieve the minimum version specified. It compares them using function XICRA.config.set_config.check_install_module().

Parameters

Debug (boolean) – True/False for debugging messages

Returns

Print messages if packages are installed.

XICRA.config.set_config.get_python_packages(Debug)[source]¶

Retrieves the version of the python packages installed in the system.

It retrieves the dependencies name conversion from file XICRA/config/python/module_dependencies.csv using function XICRA.config.extern_progs.file_list() and XICRA.scripts.functions.main_functions.get_data(). For each module it retrieves the package version installed in the system using XICRA.config.set_config.check_package_version().

Returns

Dictionary containing for each python module (key) the installed version (value).
