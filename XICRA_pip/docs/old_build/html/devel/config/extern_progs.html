XICRA.config.extern_progs.read_dependencies()[source]¶

Returns a dictionary containing the executable name for each software.

It uses XICRA.config.extern_progs.file_list() to retrieve absolute path for file XICRA/config/software/dependencies.csv. It then reads csv into pandas dataframe using XICRA.scripts.functions.main_functions.get_data() and returns it.
